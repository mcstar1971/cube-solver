# Cube Solver - AI魔方求解器

一个基于微信小程序的智能魔方求解器，通过视频扫描识别魔方状态，提供最优还原步骤和3D动画引导。

## 功能特性

- 🎥 **视频流扫描** - 实时识别魔方各面颜色
- 🧮 **最优解算法** - Kociemba算法，20步内还原
- 🎮 **3D动画引导** - 直观展示每一步操作
- 📝 **文本提示** - 清晰的操作说明
- ⏱️ **计时统计** - 记录还原时间

## 技术栈

- **前端**: 微信小程序原生框架
- **3D渲染**: Three.js (three-platformize)
- **算法**: Kociemba两阶段算法
- **图像处理**: Canvas API + 颜色识别

## 使用说明

1. 打开小程序，对准魔方
2. 按提示缓慢转动，扫描6个面
3. 点击"开始还原"
4. 跟随3D动画和文字提示操作
5. 完成还原！

## 开发环境

1. 安装微信开发者工具
2. 导入项目目录
3. 填写你的 AppID
4. 开始开发调试
